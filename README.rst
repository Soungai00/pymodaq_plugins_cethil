Instruments
===========

Below is the list of instruments included in this plugin

Viewer0D
++++++++

* **Keithley2700**: control of yyy 0D detector

Viewer1D
++++++++

* **Keithley2700**: control of yyy 1D detector


Installation instructions
=========================

* PyMoDAQ’s version: 4.0.11
